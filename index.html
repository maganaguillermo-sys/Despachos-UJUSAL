☁️ Nube — datos compartidos entre todos (Supabase)
📖 ¿Cómo creo la nube gratis? (5 minutos, una sola vez)
  1. Entra a supabase.com, crea una cuenta gratis y luego un proyecto nuevo (botón "New project").
  2. En el menú izquierdo abre SQL Editor, pega este código y presiona Run:
create table pedidos (id text primary key, data jsonb not null);
create table notifs (id text primary key, data jsonb not null);
alter table pedidos enable row level security;
alter table notifs enable row level security;
create policy "acceso_app" on pedidos for all using (true) with check (true);
create policy "acceso_app" on notifs for all using (true) with check (true);
  1. Ve a Project Settings → API y copia la Project URL y la clave anon public.
  2. Pégalas arriba y presiona Conectar.
  3. En el celular o computadora de cada vendedor, abre la app, entra a ⚙️ y conecta con la misma URL y clave. ¡Todos verán los mismos pedidos!

Nota: cualquier persona con la URL y la clave puede ver los datos. Compártelas solo con tu equipo.
